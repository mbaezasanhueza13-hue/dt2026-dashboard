⚙ Configuración inicial

Conecta este dashboard a tu planilla de Google Sheets en 3 pasos:

1
ID de tu Google Sheet

Copia el ID desde la URL de tu planilla:
docs.google.com/spreadsheets/d/→ ESTE ID ←/edit

2
API Key de Google Sheets

Obtén una key gratuita en Google Cloud Console → Credenciales → Crear clave de API. Activa la API "Google Sheets API".

3
Hacer pública la planilla (solo lectura)

En Google Sheets: Compartir → Cualquier persona con el enlace → Lector. El dashboard solo lee datos, nunca escribe.
